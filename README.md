## Description
A simple blog application built with Django and Python
## Features

- Create, edit and delete posts
- Author profile
- Post comments
- Post tags
- Related posts
- Rich text editor
- Search
- Pagination
- Email sharing
- Share posts on Twitter
- Comment moderation
- Post archives
